### 6. TypeScript — OBSERVED

  - **Real bug:** `lib/slices/gameSlice.ts(505)` — `An object literal cannot have multiple properties with the same name` (duplicate key).
  - Type mismatches: two divergent `GameTheme` types (`gameSlice` vs `theme-config`), `RefObject<T|null>` vs `RefObject<T>` in `leaderboard-display.tsx`, `undefined` not assignable in `islandGenerator.ts(97)`, `qRequiresU`/`e: unknown` issues, test-file type errors in `objectiveGenerator.test.ts`.
